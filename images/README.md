## 图片来源

您可以从以下来源下载图片：
1. 网络搜索（如百度图片、Google图片）
2. 旅游网站
3. 个人拍摄的照片

## 图片要求

- 图片分辨率建议为1200x800像素或更高
- 图片格式建议为JPEG或PNG
- 图片内容应与文件名描述的内容相符
- 确保图片版权合规

## 完成后

下载完所有图片后，运行以下命令将图片推送到GitHub：

```bash
git add images/
git commit -m "Add images"
git push
```

这样，访问者就能正常看到网页中的所有图片了。
